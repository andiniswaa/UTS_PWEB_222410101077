# Sistem Manajemen Toko Buku Sederhana

Proyek sederhana untuk manajemen toko buku berbasis web menggunakan Laravel.

## Fitur Utama
✅ Halaman Login  
✅ Dashboard statistik toko  
✅ Pengelolaan data buku  
✅ Profil pengguna  
✅ Responsive design  

## Teknologi
- Laravel 10
- Bootstrap 5
- Font Awesome
- Blade Template
